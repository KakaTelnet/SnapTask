# Aim Review Rubric

Apply the hard gates first, then score all five dimensions from 0 to 2.

## Hard Gates

An aim can be `通过: Approved` only when all five gates are satisfied:

1. **Background clarity**: names the current state, problem, or opportunity in concrete terms. A vague wish ("make it better") fails this gate.
2. **Target users identified**: names who will use or benefit. "Everyone" without qualification fails this gate.
3. **Core value stated**: explains why this matters and what changes. A feature list without value fails this gate.
4. **Boundaries explicit**: states what is NOT included. An aim without any exclusion fails this gate unless the scope is genuinely atomic.
5. **Success criteria observable**: defines a condition that can be seen or measured. "Users are happy" fails this gate.

Use these gate results:

- `已满足`: explicit and usable.
- `需要修订`: present but vague, incomplete, or weakly connected.
- `暂无法判断`: unavailable context prevents a fair judgment.

Both `需要修订` and `暂无法判断` prevent `通过: Approved`.

## Scoring

Score each dimension:

- `0`: absent, contradictory, or unusable.
- `1`: present but vague, partial, or weakly connected.
- `2`: clear, consistent, and actionable.

1. **Direction clarity**: the background and problem are stated concretely.
2. **User relevance**: target users are specific and the value to them is clear.
3. **Scope discipline**: boundaries and exclusions are explicit and defensible.
4. **Success measurability**: success criteria are observable, not subjective.
5. **Internal consistency**: all five sections support each other without contradiction.

## Result Selection

- `通过: Approved`: score 8-10, all five hard gates are `已满足`, and no unresolved contradiction remains.
- `修订: Revision Required`: the intended aim is usable, but one or more correctable gaps prevent approval.
- `驳回: Rejected`: no observable direction exists; core value is absent; the aim is internally contradictory; or a valid review would require inventing the aim.

Fatal conditions force `驳回: Rejected` regardless of score:
- No problem or opportunity is stated.
- Success criteria are purely subjective with no observable anchor.
- The aim contains contradictory statements that cannot be resolved without guessing.

## Finding Quality

Every finding must include:

```text
问题: what is missing, vague, inconsistent, or unsupported
影响: how it affects downstream planning, scoping, or success judgment
修改要求: what information or decision the aim author must add or clarify
```

Revision requirements identify the missing decision; they do not silently make it.
